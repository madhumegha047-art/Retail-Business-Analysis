# Retail Business Analysis

## Project Overview

This project analyzes the Sample Superstore dataset using Python (Pandas), Jupyter Notebook, and Power BI.

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Power BI

## Dataset

Sample Superstore Dataset

## Dashboard KPIs

- Total Sales
- Total Profit
- Total Orders

## Visualizations

- Profit by Category
- Sales by Region
- Sales Trend Over Time
- Profit by Sub-Category

## Key Insights

- Technology generated the highest profit.
- Office Supplies produced consistent profits.
- Furniture generated the lowest profit.
- West region achieved the highest sales.
- South region recorded the lowest sales.
- Copiers were the most profitable sub-category.
- Tables generated the highest loss.
